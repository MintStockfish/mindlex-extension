# Mindlex Browser Extension

Расширение для браузера с AI-переводом страниц, анализом слов и
сохранением новой лексики в обучающие модули Mindlex.

Основной целевой браузер: Chrome. Архитектуру нужно делать
максимально совместимой с WebExtensions, чтобы позже поддержать другие браузеры.

## Цель проекта

Расширение помогает читать страницы на иностранных языках прямо в
браузере:

- переводить всю страницу;
- переводить выделенный текст;
- получать подробную информацию о слове при наведении;
- сохранять слова в Mindlex-модули;
- синхронизировать сохраненные слова с аккаунтом Mindlex для
  дальнейшего изучения в карточках.

## Основные возможности

### 1. Выбор AI-модели

Пользователь сам предоставляет API-ключи.

Предварительно планируемые провайдеры:

- OpenAI;
- Gemini;

API-ключи должны храниться только в хранилище расширения. Скрипты,
которые работают внутри страницы, не должны иметь к ним прямой
доступ.

### 2. Перевод страницы

Расширение сканирует текстовые узлы страницы, группирует их в
батчи, отправляет на перевод и заменяет текст на странице, сохраняя
структуру DOM (ну хотя бы стараясь не ломать страницу).

### 3. Перевод выделенного текста

Пользователь выделяет слово, строку, предложение или абзац и
запускает перевод через:

- плавающую кнопку рядом с выделением;
- пункт контекстного меню;
- горячую клавишу.

Результат показывается во всплывающем окне рядом с выделенным
текстом.

### 4. Анализ слова при наведении

При наведении на слово расширение показывает компактную карточку:

- исходное слово или лемму;
- перевод;
- IPA;
- часть речи;
- краткое значение;
- пример использования;
- кнопку подробного анализа;
- кнопку добавления слова в Mindlex-модуль.
- возможно TTS, но это не точно

Подробный анализ должен быть совместим с текущей логикой Mindlex:
перевод, пример, IPA, произношение, части речи, синонимы, антонимы,
формальность употребления и этимология.

## Интеграция с Mindlex

Расширение подключается к аккаунту Mindlex через авторизацию.

После входа пользователь может:

- получить список своих модулей;
- создать новый модуль;
- добавить слово в существующий модуль;
- синхронизировать слова, добавленные офлайн;
- избежать дублей карточек.

Важно: сейчас в `mindlex-ai` модули и авторизация хранятся в
`localStorage`, позже планируется это поправить.

## Технологии

- TypeScript;
- Hono
- Ky
- Manifest V3;
- WXT или Plasmo;
- React для popup, options и overlay;
- Mantine;
- Zod для runtime-валидации;
- webextension-polyfill для совместимости WebExtensions;
- Vitest для unit-тестов;
- Playwright для E2E-тестов расширения.

Рекомендуемый стартовый стек: WXT + React + TypeScript.
